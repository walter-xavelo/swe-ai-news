## Overview

On February 6, 2026, TechCrunch published "Spotify changes developer mode API to require premium accounts, limits test users". Spotify is now limiting each app to only five users and requires devs to have a Premium subscription. If developers need to make their app available to a wider user base, they will have to apply for extended quota.
